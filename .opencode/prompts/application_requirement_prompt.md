Administrar reservas
Cargar Corte 
Cargar Color
Cargar Barba
Cargar Compra a un proveedor

Administrar stock y limites de faltantes segun demanda 
Grafico ganancias
Grafico compras 
Grafico gastos

La forma de comunicacion con el agente es via wspp o telegram pero siempre de forma gratuita y todo el entorno va a estar desplegado localmente en una computadora
